This driver implements the Telegram Bot API.

It allows to send/receive messages to/from Sfera through the Telegram Messenger application.
You can therefore trigger custom logic upon the receiving of messages or you can send out notifications triggered by other events.
Messages sent from Sfera can include medias too, such as images, documents, audio, etc.

To use this driver you need to create a new Telegram Bot.
Once you have created it, add a Telegram driver instance to your Sfera installation and set the token of your bot in the token configuration parameter. Optionally, specify a secret passphase in the secret configuration parameter that will be used to protect the access to the Bot.

Once it is configured, launch Sfera and check the logs to make sure the driver is connected to the bot. Then, if a passphase is configured, add your Telegram account to the trusted list of users by sending a message to the created bot saying '/addme secret_passphase'. For instance, if the passphrase you set in the configuration is "MySecret", send '/addme MySecret'.
If the operation is successful, you will get a replay saying "OK".
